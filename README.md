# My mappings 

Those mappings are heavily influenced by [Seniply](https://stevep99.github.io/seniply/).

For pair symbols that have a closing symbol (I.e. [], () and {}). Double tap is used to activate them (See [tap dance](https://docs.qmk.fm/#/feature_tap_dance) for official documentation).

To compile and flash the firmware:

```bash
qmk flash -bl dfu -kb ferris/sweep -km marcsolanadal
```
